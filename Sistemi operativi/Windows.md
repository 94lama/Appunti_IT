# Terminale
Stampare la lista di comandi utilizzabile
```sh
help
```
Trovare l'indirizzo #IP della macchina su cui si opera
```sh
ipconfig
```
Verificare le #porte aperte su un indirizzo #IP 
```sh
telnet <indirizzo_ip>
```
