# Tag
Il #tag è una stringa utilizzata per indicare una articolare funzione di un componente. 

| Tag | Descrizione |
| ---- | ---- |
| a | Utilizzato per inserire link ad altre risorse |
### a
```HTML
<a href="link">Link</a>
```
